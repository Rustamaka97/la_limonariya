## 1. Янги фичалар (нима қўшилган)

**Меҳмон QR (public, auth йўқ) — App.tsx роутлари:**
- `?call=<tableId>` → **официант чақириш** (Официант/Ҳисоб/Сув) → ходим POS'да пульс банер+беп кўради, «Бордим» ёпади. `pos.callWaiter` (public) / `activeCalls` / `resolveCall`. Дедуп бор.
- `?pay=<tableId>` → **QR-тўлов** — меҳмон ўз чекини кўради + Payme/Click тап-линк. `pos.guestBill` (public).
- `?menu=<tableId>` → **ўзи буюртма** — менюдан танлаб → столнинг заказига қўшилади (кухняга АВТО юборилмайди — официант «Отправить» босади). `pos.guestMenu` + `guestAddItems` (public).

**POS ичи:**
- **Бронь + аванс** — олдиндан жой банд (календар+ҳафта куни), банкет аванси (кассага киради, ёпилишда `avans` тўлов қатори, бекорда refund/forfeit). +45 дақ авто no-show (стол бўшайди).
- **Курс/подача** — таомга курс белгиси (cart чип 1→2→3) → кухня тикети/KDS'да «N-курс». Аддитив, юбориш логикаси бузилмаган.
- **Официант KPI/рейтинг** — Аналитика → «Официант KPI» (тушум лидерборди). `analytics.waiterKpi`.
- **Меню муҳандислиги** — Аналитика → «Таом таҳлили» (юлдуз/от/жумбоқ/ит).
- **Мижоз таниш** — Mijozlar профили: ташриф·сарф·ўрт.чек·севган таом. `finance.customers.profile`. + мижоз бириктириш/чегирма (`attachCustomer`/`detachCustomer`/`discount`).
- **История чека** — заказ амаллари timeline (audit_log). `pos.orderEvents`.
- Официант алмаштириш · чек тозалаш · ☰ бошқарув панели · очиқ-чеклар рўйхати · стол ўлчами (drag) · тўлиқ экран · PIN клавиатура.

**Скин:** CloPOS 1:1 лайоут, **яшил+олтин** (изумруд `#0e4037` + олтин `#f3b759`).
