## W1 · Скорость как люкс ⚡ (ощущение «летает»)

**Скилл фазы:** `/w1-speed` — качает карту кода (cache.ts/outbox/Pos), правила 1.1–1.5 и gate фазы.

Премиальность на 50% — это отклик. Цель: **любое касание < 100 мс визуального ответа**.

| # | Что | Вау-эффект | Усилие |
|---|-----|-----------|--------|
| 1.1 | Optimistic UI везде, где ещё нет (закрытие чека, перенос стола, обвалка submit) — мгновенный отклик, сервер догоняет | касса «щёлкает» без ожидания | M |
| 1.2 | Skeleton-состояния вместо «⏳» на всех экранах (Dashboard, Hisobot, Analitika) | нет «дырок» при загрузке | S |
| 1.3 | Переходы между табами без перезагрузки данных — `swr()` кэш уже есть, накрыть им ВСЕ read-пути | таб открывается мгновенно, данные обновляются тихо | M |
| 1.4 | Тач-эргономика официанта: крупнее тап-зоны на карте зала, долгое нажатие = быстрые действия (перенести/гости/примечание) | официанты не промахиваются в запаре | S |
| 1.5 | Виброотклик (navigator.vibrate) на ключевые действия POS: добавил блюдо, отправил на кухню, закрыл чек | «физическое» ощущение кассы | S |

**Gate:** Lighthouse perf ≥ 90 на планшете · переход таба < 200 мс · addItem→отклик < 100 мс.
