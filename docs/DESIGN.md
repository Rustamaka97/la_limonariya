## 2. Типографика
- Шрифт: мавжуд self-hosted тўплам (fonts.css) — янги шрифт қўшилмайди.
- Пул: доим `tabular-nums`, минг ажратгичи `fmt()` (масалан `150 000`).
- Сарлавҳа оғирлиги: `font-bold`, ёрдамчи матн: `text-zinc-400/500`.

## 3. Компонент қоидалари
- Радиус: карта/модал `rounded-2xl`, тугма/инпут `rounded-lg`/`rounded-xl`.
- Соя: тинч ҳолат `shadow-sm`, кўтарилган `shadow-md` (ҳовер).
- Босиш қайтарими: `active:scale-95` + `motion-reduce:active:scale-100` (доим жуфт).
- Сенсор нишони: телефонда минимал 40×40px (официант бармоғи).
- Модал: `fixed inset-0 z-50 bg-black/40`, панел `rounded-t-2xl` (мобилда пастдан) / `sm:rounded-2xl`.
- Ҳар амалга ҳолат: idle → busy («…» ёки disabled) → натижа/хато. Индикаторсиз кутиш йўқ.
