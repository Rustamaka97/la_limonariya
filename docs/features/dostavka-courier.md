# Доставка / Курьер (delivery) — backend + frontend spec

> Етказиб бериш заказлари — стол йўқ, курьер + мижоз манзили билан. La Limonariya'да ЙЎҚ (фақат зал/собой). Приоритет: **P2** (доставка қиладиган ресторон учун).

## 1. CloPOS'да (жонли — тест қилинди)
- **Новый заказ → тур:** На месте · **Доставка** · С собой. «Доставка» танланса:
  - **Стол ЙЎҚ** (сарлавҳа «Доставка», #a5da2149).
  - **Сервис 0%** (доставкада хизмат ҳақи **йўқ**; залда 10%).
  - **Мижоз + манзил:** «**Добавить клиента**» → мижознинг **манзили** (§ Клиенты #2 — `customer_addresses`, «Адреса» майдони).
  - **Курьер:** Чеки/Архивда «**Курьер**» фильтр → заказга курьер бириктирилади.
- Заказ амаллари худди залдагидек (Скидка, Разделить, комментарий...), лекин **Изменить стол / гости йўқ** (стол-сиз).

## 2. La Limonariya — ҲОЗИР
- Фақат **зал / собой** (`orders.hallId` + `halls.servicePct`; собой = 0% сервис). **ЙЎҚ:** доставка тури, **курьер**, мижоз **манзили**.

## 3. BACKEND spec
- **Схема:**
  - `orders`га `+ type pgEnum('zal','soboy','dostavka')` (ёки мавжуд hall/soboy мантиғини кенгайтириш). `dostavka` → `servicePct=0`, `hallId` ихтиёрий (стол-сиз).
  - `orders`га `+ courierId uuid → users` (курьер = янги роль `courier`, ёки мавжуд ходим).
  - **Манзил:** `customer_addresses` (§ Клиенты #2) — заказга танланган манзил (`orders.addressId?`).
- **tRPC:** `pos.create`га `type='dostavka'` + `courierId?` + `addressId?`. `pos.assignCourier({orderId, courierId})`.
- **Молия:** доставка тушуми алоҳида (byType), курьер бўйича ҳисобот (§ Чеки Курьер фильтр).

## 4. FRONTEND spec (`Pos.tsx`)
- **Новый заказ → «Доставка»** тури: стол-сиз заказ, **0% сервис**, **мижоз + манзил пикер** (Клиенты addresses), **курьер бириктириш**.
- **Floor:** доставка заказлари алоҳида рўйхат (стол харитасида эмас — «Собой/Доставка» бўлими).
- Чеки/Архивда **Курьер фильтр** (§ cheki-royxat-filtr #4) — қайси курьер қайси заказ.

## 5. Приоритет / боғлиқлик
- **P2** — доставка қиладиган ресторон учун. **Клиенты** (#2, манзил) + **Чеки** (#4, курьер фильтр) билан боғлиқ.
- Тип продажи (На месте/Доставка/С собой) — заказ toolbar'да ҳам бор (§ zakaz-amallari #12).
