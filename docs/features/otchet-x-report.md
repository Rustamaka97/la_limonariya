# Создать отчет (X-отчёт) — backend + frontend spec

> Давр бўйича сотув сводкаси (X-отчёт), принтерга босиш билан. La Limonariya'да асоси бор (`financeForWindow`/`pnl`/`dayClose`), лекин ихтиёрий оралиқ + меҳмон/йўқотиш турлари + печать йўқ. Приоритет: **P2**.

## 1. CloPOS'да (жонли, тўлиқ кўрилди)
- **Оралиқ:** «С этой даты» → «До этой даты» (сана + вақт, 30-мин). Кун чегараси **06:00** (La Limon билан бир хил).
- **Фильтрлар:** **Терминалы** (қайси терминал) + **Тип отчёта** («C операциями» — касса кирим/чиқим билан / «Счета» — ҳисоблар).
- **Натижа — «Отчет о продажах»:** Промежуточный итог · Налог · Угощение (текин) · **Сервис** · Скидка · Стоимость доставки · Возврат товара · **ЖАМИ**.
- **Другие детали:** Количество гостей · Открытые чеки · Сумма открытых чеков.
- **Способы оплаты:** Наличными (N чек) + **йўқотиш турлари**: Гость ушел · Ошибка сотрудника · За счет компании.
- Тугма: **Распечатать** (Контрольный принтер).

## 2. La Limonariya — ҲОЗИР
- **Backend (тайёр асос):** `finance.dayClose` (кун), `finance.pnl` (оралиқ), `financeForWindow` — revenue, byMethod (нақд/карта/click/payme/қарз), **сервис** (per order servicePct), cogs, opex, cardTax, guestDebt, refundTotal, comp. `till` (X/Z смена: `expectedCash` vs `countedCash`).
- **Frontend:** `Moliya.tsx` — молия сводкаси + смена (till count).
- **ЙЎҚ:** ихтиёрий **вақт оралиғи** (соат аниқлигида; ҳозир кун/сана оралиғи), **терминал фильтр**, **меҳмонлар сони** отчётда, **«гость ушел / ошибка сотрудника»** йўқотиш турлари, **принтерга X-отчёт**.

## 3. BACKEND spec (кенгайтириш)
- **`finance.xReport({from, to})`** — `financeForWindow`ни ихтиёрий (соат аниқлигидаги) оралиққа кенгайтириш + қўшимча:
  - **Меҳмонлар сони:** `SUM(orders.guests)` (ёпилган, оралиқда).
  - **Йўқотиш турлари:** Угощение = comp orders (бор); «Гость ушел» = `staleOrders`/бекор қилинган очиқ; «Ошибка сотрудника» = voided/cancelled. (Янги enum ёки мавжуд сигналлардан.)
  - Тип отчёта «C операциями»: `cash_operations` (§ Касса операциялари MD) қўшилади.
- **Печать:** натижани **станция печати мост-агентига** (Контрольный принтер) — § stansiya-pechati-bridge.

## 4. FRONTEND spec (`Moliya.tsx`)
- **«X-отчёт» бўлими:** оралиқ танлаш (сана+вақт) → натижа: сотув таркиби (жами, сервис, скидка, возврат) + тўлов усули + меҳмонлар + йўқотиш турлари → **Распечатать** (мост-агент).

## 5. Хулоса Шерхонга
- Асосан **`financeForWindow` бор** — ихтиёрий оралиқ + меҳмон/йўқотиш турлари + принтер қўшиш.
- Печать — станция печати мост-агенти билан келади. Приоритет: **P2**.
