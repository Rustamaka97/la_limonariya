# 🖨 Print-agent — термо-принтер учун

Касса компьютерида ишлайдиган кичик сервис: веб-иловадаги «Чоп этиш» тугмаси
чек/тикетни шу агентга юборади, агент ESC/POS байтларга айлантириб принтерга
узатади. Агент ишламаса ёки хато берса, веб-илова ўзи автоматик равишда одатдаги
браузер print диалогига қайтади — ҳеч нарса бузилмайди.

## Ўрнатиш (касса компьютерида, бир марта)

1. Node.js ўрнатилган бўлсин (тизимда бор).
2. `config.example.json` ни `config.json` қилиб нусхаланг ва созланг:
   - **Тармоқ принтери** (Ethernet/Wi-Fi): `"mode": "tcp"`, `printerHost` = принтер IP.
   - **USB принтер**: Windows'да принтерни **share** қилинг (Generic / Text Only
     драйвер билан), кейин `"mode": "windows"`, `"printerName": "SHARE_NOMI"`.
   - `width`: 58 ёки 80 (мм) — чек қоғози эни.
3. Ишга тушириш: `node print-agent.mjs`
4. Автостарт учун (Windows): Task Scheduler → At log on →
   `node C:\...\print-agent.mjs`.

## Текшириш

Браузерда `http://127.0.0.1:9110/health` → `{"ok":true,...}` чиқса — тайёр.

## Эслатма

- Ўзбек кирилл ҳарфлари (ў, қ, ғ, ҳ) CP866 кодировкада йўқ — чекда энг яқин рус
  ҳарфи билан чиқади (у, к, г, х). Ўқилиши бузилмайди.
- Принтер CP866 ни қўлламаса, принтер утилитасида default codepage'ни
  «Cyrillic (866)» га ўтказинг.
