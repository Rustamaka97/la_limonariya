## Лойиҳа нима

Навои шаҳридаги **La Limonariya** ресторани учун POS + омбор + аналитика тизими.
CloPOS'ни **тўлиқ алмаштиради**. Шунчаки касса эмас — **ўз-ўзини текширадиган
анализ-машина**: остаток, таннарх ва аномалияларни тарихдан ўрганиб, директорни
алдашнинг тешикларини ёпади.

## 3 та бузилмас constraint (бутун дизайн шунга бўйсунади)

1. **Offline-first** — касса/официант интернетсиз ишлайди. Барча ёзиш append-only
   outbox (UUID идемпотентлик), реконнектда flush. Маълумот йўқолмайди.
2. **Un-fakeable ҳисоб** — остаток, таннарх, баланс **ҳеч қачон қўлда таҳрирланмайди**.
   Фақат event ledger'дан ҳисобланади: `stock_movements`, `finance_transactions`,
   `customer_ledger`.
3. **Multi-branch day 1** — `branch_id` ҳар бир жадвалда. Ҳозир 1 филиал (Навои).

## Кесувчи қоидалар (ҳамма жойда амал қил)

- **Пул**: UZS **бутун сон**, `Math.round`, касрсиз. Ҳеч қачон float билан пул сақлама.
- **Вақт**: DB'да **UTC** сақла, UI'да **UTC+5** (Asia/Tashkent) кўрсат.
- **Иш-куни**: 06:00 → 06:00 (яримкечада эмас).
- **UI тили**: узбек-кирилл (uz-Cyrl).

## Буйруқлар (root'дан)

```bash
pnpm dev            # ҳамма app parallel dev
pnpm build          # ҳаммасини build
pnpm typecheck      # ⚠️ код ёзгандан кейин ДОИМ юр
pnpm db:generate    # schema.ts ўзгартирсанг — ШАРТ (миграция ясайди)
pnpm db:migrate     # миграцияни қўллаш
pnpm up / pnpm down # docker compose
```

### 🔴 ЭНГ КЎП УЧРАЙДИГАН ХАТО
`schema.ts`ни ўзгартириб `pnpm db:generate` **қилмасанг** — жадвал DB'да
яратилмайди → runtime'да **500**. (Analytics баги айнан шундан эди.)
Схема тегса → доим `db:generate` → миграцияни текшир.

## Иш услуби (Рустам ака қоидалари)

**Карпати 4 принципи:**
1. **Аввал ўйла** — код ёзишдан олдин муаммони тушун.
2. **Соддалик** — энг оддий ечим. Ортиқча абстракция йўқ.
3. **Хирургик ўзгартир** — атрофдаги кодга тегма, фақат кераклисини.
4. **Мақсадга йўналтир** — ҳар ўзгариш аниқ мақсад учун.

- Жавоблар **кирилча ўзбекча**, боевой, аниқ. Сув қуйма.
- Катта ўзгаришдан олдин — қисқа assumption ёз, тасдиқ кут.
- Атрофдаги код стилига мослаш (naming, comment зичлиги, идиома).
- **Маъқуллама — ҳақиқатни айт.** Ҳар ғоя/қарорда аввал заиф томон ва рискни айт.
  Катта қарор (пул, схема, прод) = қаттиқ танқид шарт; майда иш = тез бажар.
- **Устоз-шогирд режими.** Рустам хато/ноаниқ сўраса — ишни тўхтатма, қилиб бер, кейин
  ёнида қисқа prompt-техника эслатмаси бер (ишлатиб ўргат, лекция қилма).
