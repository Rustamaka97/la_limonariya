# 📊 Кунлик савдо — La Limonariya

## Иш-куни чегараси (МУҲИМ)
06:00→06:00 (CLAUDE.md). DB'да UTC, UI'да UTC+5. Бугунги иш-куни =
[бугун 06:00, эртага 06:00) Asia/Tashkent → UTC'да [бугун 01:00, эртага 01:00).
Аргумент сана берилса — ўша кун.

## 0. Уланиш
Postgres MCP (`limon-db`) уланганми? Йўқ бўлса — `la_limonariya`да `pnpm up` + POS оч
([[la-limon-claude-config]]). Устунларни `schema.ts`'дан тасдиқла: `orders`
(status/created_at/closed_at/discount_amount), `order_items` (product_id/price/qty/weight_g),
`order_payments` (amount/method), `voided_items`.

## 1. Жами тушум
- Ёпилган `orders` (status closed/paid) бугунги иш-кунида.
- `order_payments` SUM — жами пул, **method бўйича** (нақд/карта/Payme/Click).
- Чек сони, ўртача чек. Пул UZS **бутун** (Math.round, касрсиз).

## 2. Топ таомлар
- `order_items` JOIN `products` — qty бўйича топ-10. Дона + пул.
- Оғирлик сотилган (weight_g>0) — гўшт кг алоҳида кўрсат.

## 3. 🔴 Аномалиялар
- Чегирма: `discount_amount>0` — жами + катта (масалан >20% чек) чегирмалар, сабаби.
- Бекор: `voided_items` — сони, ким, қанча пул (алдов сигнали бўлиши мумкин).

## 4. dataviz (ихтиёрий)
Соатлик савдо графиги сўралса — `dataviz` skill қоидаларига мос SVG (dep-free).
