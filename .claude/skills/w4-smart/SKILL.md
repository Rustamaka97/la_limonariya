## Карта кода

- `apps/api/src/telegram.ts` — `sendTelegram` (секреты `TELEGRAM_*` только в `.env` сервера — **никогда в код/коммиты/видимый текст команд**).
- `apps/api/src/telegram-digest.ts` — cron-обёртка 23:00 (`docker compose exec -T api npx tsx src/telegram-digest.ts`), вызывает `sendDailyDigest` из `router.ts`.
- `docker/install-cron.sh` — сюда добавлять новые cron-строки.
- `apps/api/src/time.ts` — `businessDayBounds` (день = 06:00 Ташкент); все окна времени только через него.
- Данные уже есть: маржа блюд (`computeDishTaannarx`), COGS в digest, `stock_movements` (расход по продуктам), аномалии обвалки/цен, voids/refunds/discounts.

## Правила по пунктам

- **4.1 Инсайты из правил**: новый модуль `apps/api/src/insights.ts` — чистые функции на узбекском. Стартовый набор: (а) маржа блюда падает 3 дня подряд → «X маржаси 3 кун кетма-кет тушди — нархни кўтаринг ёки харидни текширинг»; (б) COGS% дня > медианы 4 недель на 5пп; (в) выручка пикового часа ниже прошлой недели. Подключить в `sendDailyDigest` блоком «💡 Тавсиялар». Максимум 3 инсайта за день (приоритизация), иначе спам.
- **4.2 Прогноз закупа**: расход мяса (stock_movements, отрицательные qty по мясным продуктам) за 4 недели, **медиана того же дня недели** → «Эртага жума: ~18 кг қўй, ~12 кг мол керак бўлади» в дайджест 23:00. Меньше 4 точек данных → строку не выводить вовсе. Всегда помечать «~тахминий».
- **4.3 «Кончится к вечеру»**: линейная экстраполяция расхода с 11:00 по «сейчас» vs onHand → «Қўй лаҳм соат ~20:00 да тугайди». Отдельный скрипт `apps/api/src/telegram-runout.ts` + cron `0 16 * * *` в `install-cron.sh`. Порог и список отслеживаемых продуктов — настраиваемые (settings). Анти-спам: один продукт — один пуш в день.
- **4.4 Недельные рекорды**: в воскресном дайджесте (проверка дня недели): лучший день недели, лучший официант (по выручке закрытых заказов), «🏆 рекорд тушум за всё время» при побитии максимума.
- **4.5 Аномалии человеческим языком**: `humanizeAnomaly()` — вместо флагов шаблоны: «Бозорчи 52 кг деди, обвалка 47.8 кг кўрди — 4.2 кг фарқ (8%)». Заменить сырые флаги в digest на фразы.

## Формат текста (единый)

Числа `toLocaleString("ru-RU")`, деньги «so'm», кг с 1 знаком, время `HH:mm` Ташкент. Прогнозы всегда с «~» и словом «тахминий». Тон — короткий совет, не отчёт.
