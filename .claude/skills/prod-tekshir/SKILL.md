# 🚨 Prod текшир — La Limonariya

Мақсад: прод ишга тушишдан олдин **5 рискни жонли текшириш**. Статик рўйхат эмас —
DB'нинг **ҳозирги** ҳолати. ([[la-limon-prod-launch-audit]] тарихий контекст.)

## 0. Уланиш текшируви

1. Postgres MCP (`limon-db`) уланганми? Уланмаган бўлса: Рустамга айт — `la_limonariya`да
   `pnpm up` қил + POS репони оч ([[la-limon-claude-config]] MCP қисми). Агар docker
   ишламаса — `.env`'даги `BOOTSTRAP_DIRECTOR_PIN` ва seed файлларини ўқиб текшир.
2. Устун номларини `apps/api/src/db/schema.ts`'дан **тасдиқла** (схема ўзгариши мумкин):
   `users` (role, pin_hash, pin_lookup), `stations` (ip?), `products` (type, kind),
   `recipes` (product_id?). SQL'ни ўша устунларга мослаб ёз.

## 1. 🔴 Директор PIN дефолтми (ЭНГ МУҲИМ)

- `users`'да role=`director` қаторини топ. `pin_lookup` — дефолт `1234`нинг lookup/hash'ими?
- `.env`'даги `BOOTSTRAP_DIRECTOR_PIN` ҳали `1234`ми? `auditLog`'да `pin.reset` бормими
  (алмашганини тасдиқлайди)?
- Агар дефолт — **🔴🔴 ЖОНЛИ ХАВФ** (тизим pos77 орқали интернетда очиқ). PIN'ни мен
  қўймайман — эга ўзи алмаштиради.

## 2. 🔴 Ходим PIN'лари

- `SELECT role, count(*) ... WHERE pin_hash IS NULL` — нечта ходим PIN'сиз (кира олмайди)?
- 14 ходим бор эди; ҳаммасида PIN борми текшир.

## 3. 🔴 Принтер IP'лари

- `stations`'да IP майдони бўш станцияларни топ (айниқса **NON CHOY** — охирги аудитда бўш эди).
- Карта ([[clopos-printer-map]]): SALAT `.131` · OSHXONA `.132` · SHASHLIK `.133` ·
  BALIQ `.134` · BAR `.137` · NON CHOY номаълум. IP бўш = ўша станцияга босилмайди.

## 4. 🟡 Тех-карта уланиши

- `products` type=`dish` (таом) нечтаси `recipes`'сиз (рецепт уланмаган)?
- Уланмаган = таннарх/маржа/аномалия сигналлари ухлаяпти (сотув ишлайди, "ақл" йўқ).

## 5. 🟡 Дубликат маҳсулот

- `products`'да бир хил (ёки жуда ўхшаш) номли қаторлар — `GROUP BY lower(name) HAVING count>1`.
- Тарихда: Фарш кг ×2, Гарди олма-лимон ×3, Aristokrat ×2.

## Ҳисобот формати

Ҳар рискни рангла ва **аниқ рақам** бер:
```
🔴 Директор PIN: дефолт 1234 — ЗУДЛИК билан алмаштир
🟢 Ходим PIN: 14/14 бор
🔴 Принтер IP: NON CHOY бўш (5/6 тўлган)
🟡 Тех-карта: 60/65 таом уланмаган
🟢 Дубликат: йўқ
```
Охирида: **ишга тушишни блоклайдиган** рисклар (🔴) рўйхати + кейинги қадам.
Ёзиш (IP киритиш, PIN) — эга рухсатисиз қилма.
